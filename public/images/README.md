# Библиотека изображений

Эта папка содержит все изображения для сайта Setedi.

## Структура

- `gallery/` - изображения для галереи
  - Интерьер салона
  - Релакс-зона
  - SPA-процедуры
  - Зона ожидания
  - Массажные кабинеты

- `team/` - фотографии команды
  - Анна
  - Мария
  - Елена

- `icons/` - иконки и логотипы

## Использование

Для использования изображений в компонентах используйте путь:
```jsx
<img src="/images/gallery/interior.jpg" alt="Интерьер салона" />
```
